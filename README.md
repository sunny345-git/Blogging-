Python Blogging & Social Platform

A web-based blogging and social platform built with Python that allows users to create accounts, publish posts, and interact with content.

#Features
- User authentication (signup and login system)
- Create, edit, and delete blog posts
- User interaction with content
- Secure handling of user data

# Tech Stack
- Python
- Django
- SQLite

# Project Overview
This project demonstrates my ability to build a functional 
web application with core backend features such as 
authentication,
content management, and data handling.

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the development server
